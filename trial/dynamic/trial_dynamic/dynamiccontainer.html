DynamicContainer

A DynamicContainer is a Container that stores heterogenously typed elements.

Table 1.4. Nullable semantics

Expression

Return type

Nullable semantics

Conditions

v.clear()

void

No effect.

v.erase(p)

iterator

No effect.

v.erase(i, j)

iterator

No effect.

v.insert(u)

iterator

Changes stored type to sequence and inserts u.

Effects: 

v == { u }

v.insert(i, j)

void

Changes stored type to sequence and inserts all elements in range [i, j)

v.insert(p, u)

iterator

Fails.

v.insert(p, i, j)

void

Fails.

Table 1.6. Sequence semantics

Expression

Return type

Sequence semantics

Conditions

v.clear()

void

Removes all nested elements.

Effects: 

v.empty() == true

v.erase(p)

iterator

Removes element at position p.

v.erase(i, j)

iterator

Removes all elements in range [i, j).

v.insert(u)

iterator

Inserts u at end.

v.insert(i, j)

void

Inserts all elements in range [i, j) at end.

v.insert(p, u)

iterator

Inserts u before position p.

v.insert(p, i, j)

void

Inserts all elements in range [i, j) before position p.


Table 1.7. Associative semantics

Expression

Return type

Associative semantics

Conditions

v.clear()

void

Remove all nested elements.

Effects: 

v.empty() == true

v.erase(p)

iterator

Remove element at position p.

v.erase(i, j)

iterator

Remove all elements in range [i, j).

v.insert(u)

iterator

Inserts u.

Requires: 

u is a pair.

v.insert(i, j)

void

Inserts all elements in range [i, j)

Requires: 

Each elements in range [i, j) is a pair.

v.insert(p, u)

iterator

Inserts u with position p as hint.

Requires: 

u is a pair.

v.insert(p, i, j)

void

Inserts all elements in range [i, j) with position p as hint.

Requires: 

Each elements in range [i, j) is a pair.
