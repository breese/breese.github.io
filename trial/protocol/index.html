Introduction

Trial.Protocol is a header-only library that provides parsers and generators for network wire protocols. Currently supported protocols are:

  • JSON

The basic building-block for each protocol is an incremental parser (or pull parser) and an incremental generator (or push formatter.) These can either be used directly, or together with Boost.Serialization.

Trial.Protocol serialization relies on Boost.Serialization, which is not header-only.

The protocol generators can write the encoded output to different types of buffers. The correct header files must be included for this to work seamlessly.

Note

For brevity all examples in this documentation assumes 

using namespace trial::protocol;
